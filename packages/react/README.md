# Установка

```shell
npm i @otmozorok/react
```

## Как использовать

```js
import { createRoot } from 'react-dom/client';
import { Avatar } from '@otmozorok/react';

const root = createRoot(document.querySelector('body'));
root.render(<Avatar>BH</Avatar>);
```
